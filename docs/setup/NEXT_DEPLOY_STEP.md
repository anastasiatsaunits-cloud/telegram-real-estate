# Следующий шаг для открытия из Telegram

Чтобы mini app реально открывалась из Telegram как WebApp, нужен публичный HTTPS URL.

## Что нужно сделать дальше
1. поднять публичный домен или временный туннель
2. направить его на `apps/miniapp`
3. прописать URL в `TELEGRAM_WEBAPP_URL`
4. запустить бота с этим URL

## Временные варианты для разработки
- ngrok
- Cloudflare Tunnel
- любой публичный reverse proxy

## После этого можно
- открыть mini app из кнопки бота
- проверить flow уже почти как боевой
- дальше подключать Telegram WebApp SDK
