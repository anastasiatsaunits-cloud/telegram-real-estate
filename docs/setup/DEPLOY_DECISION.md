# Важное решение перед боевым открытием miniapp

## Уже можно деплоить на Vercel
Miniapp готова к deploy.

## Но есть важная зависимость
Miniapp сейчас ходит в backend API.

Значит для реального публичного открытия из Telegram нужно два публичных адреса:
1. публичный URL miniapp
2. публичный URL backend API

## Иначе что произойдёт
Если задеплоить только miniapp, а backend оставить локальным:
- страницы откроются,
- но живые данные, лиды и подборка работать не будут.

## Правильный боевой путь
1. деплой miniapp
2. деплой backend API
3. прописать `NEXT_PUBLIC_API_URL`
4. прописать `TELEGRAM_WEBAPP_URL`
5. тест из Telegram
