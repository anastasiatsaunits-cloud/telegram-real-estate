# Bot welcome reference

## Что должен чувствовать пользователь
- не технический MVP
- не просто чат-бот
- ощущение закрытого каталога и полезного ассистента
- вход через красивый оффер, а не сухой utility text

## Структура welcome flow
1. Короткое сильное welcome сообщение
2. Раскрытие ценности списком
3. CTA-блок с выбором:
   - Подобрать объект
   - Открыть каталог

## Тон
- премиальный
- уверенный
- короткий
- без слов `miniapp`, `backend`, `MVP`

## Текущий рабочий текст
### Сообщение 1
🔐 Доступ к каталогу недвижимости открыт.

Этот бот поможет быстро найти объекты для жизни, инвестиций и сохранения капитала.

### Сообщение 2
💎 Что внутри:
— подборки по Крыму и Сочи
— актуальные предложения под разный бюджет
— вход в подбор по вашей цели: жить, инвестировать, сохранить капитал
— быстрый переход к объектам и заявке

Всё собрано в одном месте, чтобы не искать варианты вручную.

### Сообщение 3
👇 Выберите, как удобнее начать:

## Следующий визуальный шаг
Сделать welcome image/banner:
- тёмная палитра
- карточки объектов
- золотой/светлый CTA
- визуально ближе к референсам конкурента

## Ограничение на 2026-04-19
Попытка генерации banner через image_generate упёрлась в quota/billing limits, поэтому нужен локальный дизайн-артефакт или повторная генерация позже.
