# Visual direction — Telegram Real Estate Miniapp

## Product feeling
- private access
- luxury investment service
- calm confidence
- premium concierge tone
- soft cinematic atmosphere
- seamless mobile-first experience

## Emotional target
The app should feel not like a catalog, but like entry into a private investment club.
Not aggressive luxury, but restrained expensive confidence.

## Core visual principles
1. Full-screen feel instead of boxed website feel
2. Depth, atmosphere, glow, texture, not flat UI
3. Strong hero-image-led first impression
4. Card composition should feel editorial, not generic marketplace
5. Mobile UI should feel tactile, elegant, and expensive
6. Motion should be soft and premium, not flashy

## Anti-patterns
- empty white space that feels like unfinished layout
- generic marketplace cards
- weak hero without emotional impact
- plain blocks without atmosphere
- cheap gradients or overdecorated animations
- standard landing-page feeling

## Moodboard categories
- premium sea-view interiors
- soft luxury architectural photography
- calm cinematic skyline and coastline shots
- private club / concierge mood
- editorial real estate presentation style
- black / deep green / warm stone palette

## UI direction
- soft dark / premium neutral base
- layered backgrounds
- high-end typography hierarchy
- elegant CTA buttons
- subtle glow / blur / transparency where justified
- premium cards with strong imagery and restrained overlays

## Priority screens
1. Hero / home screen
2. Market cards block
3. Property card preview
4. Quiz screens
5. Detail page transitions

## Definition of success
If opened on iPhone, the first reaction should be:
"this feels expensive, calm, private, and beautifully directed"
not
"this is a neat prototype"
