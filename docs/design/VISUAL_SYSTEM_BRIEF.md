# Visual system brief — Telegram Real Estate Miniapp

## Palette direction
- near-black base
- deep green premium accent
- warm stone / sand neutral
- soft ivory typography contrast
- subtle metallic warmth only as accent, not dominant gold

## Surface rules
- avoid flat white page background
- use layered dark-neutral surfaces
- cards should sit inside atmosphere, not on empty canvas
- use blur / transparency only where it adds depth

## Typography direction
- strong elegant hero headline
- premium readable sans for UI
- possible serif accent only in controlled hero/detail moments
- generous spacing, never dense luxury cliché

## Card rules
- stronger imagery
- restrained overlays
- clear hierarchy: market / promise / CTA
- expensive but calm border and radius logic

## CTA rules
- primary CTA should feel premium concierge, not marketplace button
- no cheap bright fills
- clean contrast and tactile press states

## Spacing and rhythm
- more vertical breathing room
- fewer boxed fragments
- sections should flow, not stack like generic landing blocks

## Mobile-first constraints
- every hero crop must survive iPhone framing
- text overlays must stay readable on mobile
- safe areas should feel intentional
- scroll rhythm should feel smooth and cinematic
